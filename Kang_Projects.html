Project: Scratch Interactive Story

Course: CSE (Computer Science and Engineering)

Duration: August 2017 - September 2017

Objective: Create an interactive story using Scratch

Type of Project: Pair Coding


For the Scratch Project, my partner, Yoga Kanneboina, and I had to first come up with a story idea that was both interactive and interesting. The requirements were that the story be user-interactive and to have different paths that the story could go down depending on how the user responded to the prompts on the screen. Previously, we had been given an assignment that taught us how to use Scratch to create a simple game. We also had another assignment that taught us some Scratch animation basics that we used to create a simple story.Our interactive story had multiple riddles that progressively got harder as the player advanced through the game. Throughout the game, the player has to use their arrow keys on the keyboard to maneuver the character around the screen in order to "touch" the object that is the answer to the riddle. The theme of our project was an Escape Room, and the main objective of the game was to make it out of the mansion that you start in.


Project: Python Image Manipulation Project

Course: CSE (Computer Science and Engineering)

Duration: August 2017 - September 2017

Objective: Create a program that manipulates a folder of images all at once for a client

Type of Project: Collaborative Coding


For the Python Image Manipulation Project, my partner, Anika Chekuri, and I had to first choose a client to create a program for. We chose Norsled as our cause, and strived to create a program that would make all of their pictures uniform and look professional. The requirements were that we had to cater the customer and provide a program that would satisfy their needs. We were graded according to this . Previously, we had been given many assignments, such as , that taught us the basics of image manipulation using PIL and NumPy. These assignments created results that can be seen here. Our image manipulation program resized all of the images to be uniform, added the Norsled logo to the bottom left corner, and added a navy blue frame around the images. The user could change the parameters of various methods in order to change the border color, border thickness, logo placement, logo size, image size, and image dimensions. We also created a slideshow to showcase our image manipulations.
